# 一位极简主义者的早餐

## 文稿

我，是一位极简主义者。我吃的早餐很简单。

每天早上起床，我会一边用水壶，煮上 2 个鸡蛋，一边去洗漱。

洗漱结束，我拿上鸡蛋，走路去上班。

今天，我发现鸡蛋温度比较低，可能洗漱时间太长了。

我检讨自己，以后洗漱时不能玩手机。

在路上，我敲开了鸡蛋，蛋清流了我一手。

原来，我把鸡蛋放进水壶，忘记打开开关了。

这个故事有什么启示？

嗯，真正的极简主义者，应该不吃早餐。
