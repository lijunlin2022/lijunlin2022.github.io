# 青蛙和蝎子的故事

## 文稿

一只蝎子想要过河，但是它不会游泳。

这时，它看到一只青蛙也要过河，于是请求青蛙：“好心的青蛙呀，你能背我过河吗？。

青蛙看着蝎子的毒刺，拒绝道：“你要是蛰我，我怎么办？”

蝎子回答：“我为什么要蛰你呢？我把你毒死了，我也会被水淹死的。”

青蛙觉得很有道理，于是答应了它。

行至中途，青蛙被蝎子蛰了。青蛙疑惑：“你为什么要蛰我呢？现在好了，我们都要死了。”

蝎子说：“我也不想蛰你，但我控制不了，这是我的天性。”

初次听到这个故事，我不懂其中的深意，只是觉得很好笑。

逐渐地，我发现自己在生活中，看到不少蝎子。

有这样一类人，他们犯错，你去帮助他，他们不仅不会改，反而会拖你下水，你何尝不是青蛙呢？
