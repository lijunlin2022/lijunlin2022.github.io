# 正确的目标以及正确的方法

## 什么是「正确」？

开始文章之前，我需要明确两个概念：「可信度」和「可行性」。

「可信度」和「可行性」完全是两回事。

- 一件事情的「可信度」高，是指当事人没有隐瞒信息。
- 一件事情的「可行性」高，是指倾听者容易复刻成功。

如果我研究的是历史，那么只需要「可信度」高，就是「正确」的。

但我寻找的是「人生哲学」，那么只有「可信度」和「可行性」都高时，才是「正确」的。
