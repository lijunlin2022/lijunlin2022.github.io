# 如何在艰苦的环境中生活

## 饥荒

大学时，我迷上了一款游戏 ——《饥荒》。这游戏剧情很简单，我扮演一位穿越到危险世界里的科学家，努力求生并伺机寻找回到愿世界的办法。

一般来说，玩游戏应该使人快乐，但玩《饥荒》常常让我悲愤。这主要是因为游戏危险太多，又不能存档，死了就只能重来。

被黑暗中的怪物杀死、被饿死、被猎犬咬死、精神掉光了被怪物杀死、被冻死、被猪人打死、被蜜蜂蛰死、被企鹅咬死。

很长一段时间，我都熬不过冬天，最长的一次，我有幸看到冬季的 Boss —— 一只独眼巨鹿，巨鹿给了我两爪子，我就死掉了。

## 攻略

搞得这么惨，我肯定是要查攻略，看看这个游戏怎么玩，结果呢？攻略我查了不少，都很有道理，比如让我追着牛群，牛粪可以生活、施肥，有野狗过来时往牛哥身边一躲，牛哥就会教野狗做狗的道理。也有的人要我追着猪哥，猪哥心善，喂几块肉就可以被收买，然后帮忙砍树、打怪等等。

这些技巧都很有道理，但正是因为技巧太多了，反而让我不知道该怎么做，而且照着攻略玩游戏，也没有什么意思。

后来我发现一个诀窍，利用这个诀窍，我不仅活过了冬天，还活过了春天，直到夏天被热死。

## 诀窍

什么诀窍呢？其实很简单——总是保持最佳状态。

游戏人物会饿、会掉血，还会精神状态不好。

最开始时，我偶尔会攒着食物，想着等锅造好了之后吃个饱，要么就是觉得，现在吃食物、加血太浪费了。

这样会出现一些问题，那就是「永远会有突发情况」，比如你穿越沙漠，突然就窜出来一群狗过来咬你。

于是你拼了命的到处跑、或者勇敢地与一群狗决斗，结果不知不觉间，你就饿得掉血了，之后就死掉了。

## 体会

我已经开始工作了，现在我还是经常玩饥荒，我觉得现实世界其实和饥荒世界很像。很多时候我们都会遇到突发事件，比如突然失去工作，或者亲人突然重病，又或者就像新冠疫情，突然生活就发生了天翻地覆地改变。这时候查攻略有用吗？自然是有用的，能够解决一时的问题。

但很多时候，我们其实是需要保持着自己的最佳状态，这样才能在危险来临时，真正有实力和精力去处理它。
