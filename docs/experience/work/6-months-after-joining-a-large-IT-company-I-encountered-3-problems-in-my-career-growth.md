# 一个 00 后大厂前端 2022 年终感悟

「回顾2022，展望2023，我正在参与[2022年终总结征文大赛活动](https://juejin.cn/post/7172462429929111559 "https://juejin.cn/post/7172462429929111559")」

2022 年，我大学毕业，入职美团，成为了一名前端工程师。

马上 2023 年了，我总结今年的经历，主题就是一个字——「苟」。

## 1. 校园时光——平稳结束

22 年上半年时，我已经和公司签订了三方协议。那时我所求就是安稳毕业，不出什么岔子，因此毕设做得很认真。

当时，导师希望我能结合人工智能和前端技术，做一些代码自动生成的探索。

我做了三个 Demo，简单解释了前端是如何生成代码的。

- 低代码 Demo（类似 Dreamweaver）
- 模板引擎 Demo（类似 EJS）
- 静态网页生成器 Demo（类似 VuePress）

令我为难的是，我不熟悉人工智能，虽然恶补了相关知识，但要结合人工智能和前端技术做出实物，还是力不从心。最后，我只能在论文中阐述了自己的想法。

所幸导师人很不错，加上我态度认真、工作量足，因此平稳地苟过了答辩，拿到了毕业证，成功毕业。

## 2. 工作时光——充实忙碌

21 年时，我在知乎实习，做 B 端的工作，用 React + Ant Design + Echarts 做一些图表、表单等。

22 下半年，我入职美团后，主要做 C 端的工作。技术栈大致分为三块：微信小程序、React Native 和 Vue。

我做的是 C 端的工作，后半年大致工作就是，和老员工一起评审需求、编写代码、然后测试沟通加上线。

组内技术栈比较杂，我这半年里做了 3 个 Vue 相关的 H5 需求，4 个 React Native 相关的需求，以及 3 个小程序相关的需求。

所幸我的 JavaScript 基础还不错，虽然在 Vue、React Native 和小程序中来回切，但没有多少不适应。

## 3. 业余时光——集中精力

不知道有多少人像我这样的，精力怪兽。

- CSDN
- 博客园
- 知乎
- GitHub
- 语雀
- 个人网站
- 微信公众号
- 微信小程序

我做出这个决定受多个因素影响。

第一个是公众号有评论限制，新开的公众号不能开评论。
第二个是微信小程序 8 月份开始云开发收费了。
第三个是语雀开始对个人用户收费。



这些实在是太耗费精力了，最终决定只留下两个
