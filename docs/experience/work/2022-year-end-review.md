# 从大学到职场，我在沉淀中积蓄力量

「回顾2022，展望2023，我正在参与[2022年终总结征文大赛活动](https://juejin.cn/post/7172462429929111559 "https://juejin.cn/post/7172462429929111559")」

2022 年 12 月 31 日，在新年的倒计时下，我准备总结过去一年。作为一名 00 后，处在 20 多岁的年纪，我原本想把这篇总结写得更「意气风发」些，但尝试许久，始终不能如愿。

想来也是，过去一年大环境不好，大家都在说互联网的寒冬已至，我也是处处求稳，自然没有「勃勃生机」的感觉。也罢，就谈谈我过去求稳的一年吧。

## 最后的校园时光

最后的校园时光里，我的毕业设计做得很认真。这倒不是因为我醉心学术、痴迷科研，而是因为我此时已经和美团签了三方协议，不想在毕业设计上出什么岔子。

当时，导师希望我能结合前端知识和人工智能，做一些代码自动生成的探索。

作为一名前端工程师，我很熟悉前端知识。但对于人工智能，我一无所知。要结合人工智能和前端，我力不从心。

我恶补了一段时间的人工智能（~~毕业后就全忘了~~），用苦涩乏味的语言，在毕业论文上写下了我苍白无力的思考。

考虑到人工智能部分，深度和广度肯定不够，我在前端部分多下了功夫，做了三个前端 Demo：

- 低代码（乞丐版的 Dreamweaver）
- 模板引擎（乞丐版的 EJS）
- 静态网页生成器（乞丐版的 VuePress）

用这三个 Demo，我解释了生成前端代码的方法和原理。原本浅薄的毕业论文，也算有了些深度和广度。

没有多少波澜，我如愿以偿地安稳毕业。毕业时我的情绪很微妙，不怎么伤心，也不怎么怀念。

## 初入职场

我曾经在知乎实习过，做的是 B 端的工作，具体来说就是用 React + Ant Design + Echarts 做一些图表、表单等。

有了实习经验，我原本以为自己真正工作时，能够得心应手，可实际的工作比我预想要复杂。

我在美团做的是 C 端的工作。技术栈比较杂，大致分为三大块：微信小程序、React Native 和 Vue。不过我有不错的 JavaScript 基础，也写过 Vue、React 和微信小程序，所以可以啃啃老本，上手工作速度还蛮快。

**但是知识可以学习，经验却只能积累**。

我跟着几位前辈，做了两三个微信小程序的需求，三四个 React Native 需求，才逐渐熟悉了埋点、联调、测试、打包、灰度、回滚等各种流程。需求评审时，我虽然在旁边倾听，但也只是囫囵吞枣。

在这期间，我无法解决的问题，都是前辈们兜的底。每当这时，我都会暗暗把这些问题总结整理，以免下次再犯。**能力可以暂时没有，态度必须摆正**。

所幸我也还算争气，2022 年结束了，没出啥大问题，算是苟过了初入职场的新手期。

## 业余爱好做减法

我在大学时精力很充沛，开了很多坑，包括但不限于：

1. 购买域名、服务器，搞个人网站
2. 注册微信公众号、语雀、知乎、简书、CSDN、博客园、掘金
3. 开通微信云开发，搞个人的微信小程序

过去一年，一些事情的出现改变了我的想法。8 月时，微信云开发调整的收费策略；11 月时，语雀调整了收费策略；12 月时，我的域名和服务器又需要续费了。

我一琢磨，自己挖坑太多，精力和财力被浪费了。于是我削减了自己的业余爱好：注销了域名，不再续费服务器，不再运营微信公众号、语雀和知乎，也不再搞云开发的小程序。

但个人积累还是要做的，仔细思索后，我翻出了 VitePress，捣鼓捣鼓后部署到 GitHub Pages（以后记笔记就全靠它了）。另外，我把积灰的掘金账号翻了出来，毕竟掘金是少数我写文章还有人看的社区。

## 展望 2023

一般来说，一个职场菜鸟变成老手，需要两到三年的时间。2023 年，我希望自己能够戒骄戒躁，认真积累经验。在此基础上，如果我还能写优雅的代码，更别致的文章，那就再好不过了。
